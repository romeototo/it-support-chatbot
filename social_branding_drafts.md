เพิ่งพัฒนาโปรเจคใหม่เสร็จครับ! 🎉 กับ **"IT Support AI"** แชทบอทอัจฉริยะที่เกิดจาก Pain Point คลาสสิกของชาวไอที: _การตอบคำถามเดิมๆ ซ้ำๆ ทุกวัน_ 😅

ผมเลยรวบรวมปัญหาด้านไอทีที่เจอบ่อยกว่า 200 เคส (เช่น ลืมรหัสผ่าน, ต่อ WiFi ไม่ได้, เครื่องพิมพ์ค้าง, ตั้งค่า VPN) จัดเป็น 45 หมวดหมู่ แล้วเขียนเป็น Web Application แบบเบาๆ (Zero dependencies)

**ความท้าทาย & สิ่งที่ได้เรียนรู้ในโปรเจคนี้:**
✅ **AI Integration:** ผมเชื่อมต่อกับ `Google Gemini API` เพื่อให้บอทเข้าใจบริบทของคำถามพนักงานได้ดีกว่าการค้นหาด้วย Keyword ธรรมดา
✅ **Fallback System:** ถ้า API มีปัญหา ระบบจะสลับไปใช้ Keyword-based Search อัตโนมัติ เพื่อให้ระบบไม่ล่มและช่วยเหลือผู้ใช้ได้ตลอดเวลา
✅ **UX/UI Design:** ออกแบบด้วยแนวคิด Glassmorphism คลีนๆ ให้ความรู้สึกเป็น Enterprise Tool ที่น่าใช้งาน

เป้าหมายของระบบนี้คือการทำ "Self-Service L1 Support" ช่วยแก้ปัญหาเบื้องต้นให้พนักงานได้ทันที และลดปริมาณ Ticket ที่ไม่จำเป็นให้ทีม IT มีเวลาโฟกัสกับงานโครงสร้างระบบที่ซับซ้อนมากขึ้นครับ 💻🚀
