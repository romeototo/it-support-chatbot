IT Support Chatbot

คู่มือการใช้งาน - ถามปัญหา IT ได้ทันที!

Prototype v1.0
🚀 วิธีเปิดใช้งาน
1
เปิด Terminal บนเครื่องคอมพิวเตอร์
2
พิมพ์คำสั่ง: cd ~/it-support-chatbot
3
รัน Chatbot: python3 chatbot.py
4
เสร็จ! เริ่มพิมพ์คำถาม IT ได้เลย 🎉
💬 ตัวอย่างการใช้งาน
🧑
WiFi เชื่อมต่อไม่ได้
🤖
เชื่อมต่อ WiFi ไม่ได้ ทำยังไง?

1. ตรวจว่าเปิด WiFi แล้ว
2. Forget network แล้วเชื่อมต่อใหม่
3. Restart เครื่อง
4. ติดต่อ IT Support ext. 1234
🧑
ลืมรหัสผ่าน
🤖
ลืมรหัสผ่าน ทำยังไง?

1. กด Forgot Password
2. ติดต่อ IT Support ext. 1234
3. รหัสต้องมี 8 ตัว+ตัวใหญ่+ตัวเล็ก+ตัวเลข
⌨️ คำสั่งใน Chatbot
พิมพ์ ทำอะไร
คำถาม IT ถามได้เลย เช่น "คอมช้ามาก"
help ดูหมวดหมู่ทั้งหมด
quit หรือ exit ออกจากโปรแกรม
